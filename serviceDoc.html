1. BPEL service: AutoCheck

This BPEL service aims to validate driver license and address of a candidate with simple logic. It returns 'true' iff the input string contains '0' or 'a', otherwise it returns 'false'. Both input and output are string. Notice that it can accept only one input string each time, which means if the driver license is not valid you need to send another request to check the address.

Server: Apache ODE v1.x, Apache Tomcat 6

WSDL link: http://localhost:6060/ode/processes/AutoCheck?wsdl

Sample request: http://localhost:6060/ode/processes/AutoCheck/process?input=halo

2.2.2 XSL

  • job2Rss.xsl extracts job information from rssquery.xml. Running command on terminal: java -jar saxon9he.jar rssquery.xml job2Rss.xsl keyword=Java > job2RssResult.xml

  • job2RssSearch.xsl extracts job alerts by keyword from rssquery.xml. Running command on terminal: java -jar saxon9he.jar rssquery.xml job2RssSearch.xsl keyword=Java > job2RssSearchResult.xml

  • job2RssSearchSort.xsl extracts job alerts by keyword from rssquery.xml and the results are sorted by job title/positon before being returned. Running command on terminal: java -jar saxon9he.jar rssquery.xml job2RssSearchSort.xsl keyword=Java > job2RssSearchSortResult.xml
